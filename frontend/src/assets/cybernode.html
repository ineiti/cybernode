Welcome to CyberNode

Share your harddisk, bandwidth, and CPU and decide with whom you want to share.

This is a proof-of-concept of a decentralized sharing system, which merges torrent, tor, ipfs, and a reputation system.

This implementation is a centralized version to test out the frontend of the application. The actual decentralized version where the browser is an active node is being developed here: Fledger with a simple demonstrator here: Web demo.

You can find the about page with more information.

With the current version of CyberNode you can do the following:

  • Participate in the (simulated) network and see the other nodes (test and real ones)
  • Visit webpages on the decentralized system
  • Get Mana (tokens) while you're connected and more while you share other webpages
  • Use the tokens to store a webpage in the network

Some of the possible next steps are:

  • Pages owned by users:
    • Upload by drag-and-dropping (should support at least Hugo pages)
    • Simple built-in web editor
  • Create a chat which is linked to a matrix room
  • Start merging with fledger: instead of simulating the nodes in a central server, the nodes would be really done by the browser (or server)
  • Add a reputation system where you can up/downvote other people's webpages, to define which ones you're OK of hosting and which ones you prefer to not host.
